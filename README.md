# Credit Risk Assessment with Uncertainty-Aware Decision Making and Human Escalation

## Setup Instructions

### macOS/Linux
1. Run the setup script:
   ```sh
   bash setup.sh
   ```

2. Activate the virtual environment:
   ```sh
   source uom_venv/bin/activate
   ```

### Windows
1. Run the setup script:
   ```bat
   setup.bat
   ```

2. Activate the virtual environment:
   ```bat
   uom_venv\Scripts\activate
   ```

The setup scripts will create a virtual environment and install all required packages from `requirements.txt`.
